# DataGEMS Query Recommender Api

This is the documentation site for the DataGEMS Query Recommender Api. The service is part of the wider [DataGEMS](https://datagems.eu/) platform. This is part of the overall [platform documentation](https://datagems-eosc.github.io/).

The Query Recommender Api service recommends 3 possible next queries to the user based on their current query and past behavior. 

You can use the menu options on the left to navigate through the available documentation. You may be interested to jump directly to an [Architecture Overview](architecture.md), read about some [Api Constructs](api-overview.md), investigate how to interpret [Status & Error Codes](error-codes.md) or jump straight in to the [OpenAPI Reference](openapi.md). You can find the repository for the service here: [code repository](https://github.com/datagems-eosc/query-recommender).

For any questions, you can consult the [FAQ](faq.md), check if there is a relevant [issue](https://github.com/datagems-eosc/query-recommender/issues) answering your question or contact the [DataGEMS Help Desk](https://datagems.eu/contact-us/).
